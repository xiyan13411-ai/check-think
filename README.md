# 心愿积累 App — 愿望拼图存钱

每存一笔钱，愿望就拼回来一块。

一个情绪价值型存钱工具，通过拼图、动画、文案和成就反馈，让"存钱"变得像"收集愿望碎片"一样有爽感。

---

## 当前状态

MVP 已完成，进入 **小红书爽感视觉升级阶段**。

主视觉已从抽象拼图格子升级为 **愿望物品拼装视图**（CSS 手机模型）。

## 愿望物品主视觉

核心组件 `WishObjectView` 将原本 8×5 抽象格子升级为:

1. **CSS 手机模型** — 深色机身、圆角屏幕、摄像头小孔、底部 Home Bar
2. **5×8 碎片网格** — 模拟手机屏幕内部组件，已解锁显色，未解锁半透明
3. **碎片 Pop-in 动画** — 新增碎片从远方旋转飞入目标位置
4. **能量光点** — 小金额存钱时，光点从屏幕外飞入手机内部
5. **手机 Warm Glow** — 未解锁碎片时整机发出粉色光晕
6. **玻璃反射效果** — 屏幕表面斜向渐变模拟玻璃质感

当前目标物品固定为 **抽象智能手机**，不使用任何 Apple / iPhone 官方素材。后续可扩展为自定义目标图片。

### Edge Android 桌面快捷方式

在荣耀 Magic7 Pro + Edge Android 上测试时，如果"添加到手机"后桌面没有出现图标，需要检查 Edge 是否拥有"创建桌面快捷方式"权限。

开启权限后，桌面快捷方式可以正常创建。从桌面打开后不会显示浏览器导航栏，体验接近轻量 App。
