# 心愿积累 App — 愿望拼图存钱

每存一笔钱，愿望就拼回来一块。

一个情绪价值型存钱工具，通过拼图、动画、文案和成就反馈，让"存钱"变得像"收集愿望碎片"一样有爽感。

---

## 技术栈

- **Next.js** 15 (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Framer Motion**
- **localStorage**（数据持久化）

---

## 本地运行

```bash
npm install
npm run dev
```

然后浏览器打开 `http://localhost:3000`

---

## MVP 功能

- ✅ 默认愿望目标（¥10,000 新手机基金）
- ✅ 快捷金额存钱（¥10 / ¥50 / ¥100 / ¥500）
- ✅ 自定义金额存钱
- ✅ 进度百分比 + 进度条
- ✅ 40 块拼图碎片，随进度解锁
- ✅ Framer Motion 动画（碎片飞入、按钮点击、金额高亮）
- ✅ 随机鼓励文案
- ✅ 历史记录（存钱小票）
- ✅ 成就弹窗（25% / 50% / 75% / 100%）
- ✅ localStorage 持久化，刷新不丢
- ✅ 移动端优先设计
- ✅ PWA Manifest / 图标就绪

---

## 安卓优先说明

- 优先适配 360px - 430px 宽度
- 按钮高度 ≥ 44px
- 数字输入键盘（inputMode="numeric"）
- 禁止横向滚动
- 支持设备震动反馈
- safe area 预留
