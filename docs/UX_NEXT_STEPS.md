 # 体验优化路线（下一轮布局重构）
 
 当前首页布局仍然偏工具页，与小红书参考图的"大主视觉+克制UI"有差距。
 
 ---
 
 ## 当前布局问题
 
 1. **GoalCard 占据顶部大量空间**
    - 金额数字很大，但视觉上抢了主视觉的风头
    - 建议合并进 Hero 底部的 glass panel
 
 2. **主视觉区域偏小**
    - 手机图在卡片内居中，两侧有较多留白
    - 建议让 Hero Scene 更宽，更接近"手机产品图"的感觉
 
 3. **SavePanel 与主视觉分离**
    - 存钱按钮在 Hero 下方，视觉上像两个独立区域
    - 建议把"存一笔"大按钮放在 Hero Scene 底部 glass panel 内
 
 4. **历史记录占比较重**
    - 历史记录列表占用了较多垂直空间
    - 建议折叠为"最近一条"或可展开的简要区块
 
 5. **页面整体偏功能页**
    - 元素从上到下依次排列，不像一张"心愿罐首页"
    - 缺少社交/分享所需的截图感
 
 ---

 ## 关键改动
 
 | 改动 | 说明 |
 |------|------|
 | 移除独立 GoalCard | 合并进 Hero Scene 底部 |
 | Hero Scene 加宽 | 减少两侧 padding |
 | 存钱按钮内嵌 | 放在 glass panel 底部 |
 | 历史折叠 | 只显示最近一条，点开看全部 |
 | 截图感 | 主视觉区域适合截屏分享 |
 
 ## 时机
 
 当前阶段暂不动布局，等视觉资产就位 + 主视觉定稿后再统一重构。
