# check-think docs

这个目录用于存放「愿望拼图存钱 App」的产品与开发文档。

## 文件说明

- `PROJECT_BRIEF.md`：项目书，说明产品定位、MVP 范围、功能设计、技术设计和验收标准。
- `ROADMAP.md`：完整规划路线，按阶段拆解从 MVP 到 PWA、Android、iOS 和增强版的路线。
- `CODEX_EXECUTION_PLAN.md`：给 Codex 的开发执行说明，包含推荐开发顺序、组件拆分、测试清单和最终 Prompt。

## 当前推荐执行顺序

1. 先读 `PROJECT_BRIEF.md`
2. 再读 `ROADMAP.md`
3. 最后让 Codex 按 `CODEX_EXECUTION_PLAN.md` 开发

核心原则：

> 每存一笔钱，愿望就拼回来一块。
