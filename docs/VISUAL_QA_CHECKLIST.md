 # 视觉验收清单
 
 ## 手机物品质感
 
 - [ ] 第一眼看更像产品海报，不像网页组件
 - [ ] 手机背面有真实感（镜头、金属边、背板渐变）
 - [ ] 有材质高光和阴影
 - [ ] 不出现 Apple/iPhone 官方素材
 
 ## 碎片飞回动效
 
 - [ ] 碎片看起来像手机的一部分，不是装饰方块
 - [ ] 碎片从外部飞入时轨迹自然（linear 或弹簧曲线）
 - [ ] 解锁碎片时能看到"碎片被召唤回来"的因果感
 - [ ] 未解锁碎片在外部漂浮时，有前后景层次
 - [ ] 碎片数量不让人眼花（最多 14 块）
 
 ## 100% 完成态
 
 - [ ] 手机完整显示
 - [ ] 完成时不靠 emoji 表达（使用 CSS 星点/光晕替代）
 - [ ] 有柔和光晕或呼吸效果
 - [ ] 庆祝文案不抢主视觉
 
 ## 进度反馈
 
 - [ ] 下一块碎片进度条清晰可见
 - [ ] 小金额存钱有柔和反馈（光点飞入或手机微亮）
 - [ ] 大额解锁时有较强奖励感
 
 ## 安卓性能
 
 - [ ] Android Chrome 动画不掉帧
 - [ ] 页面不横向滚动
 - [ ] 按钮尺寸 ≥ 44px
 - [ ] 页面加载不白屏
 
 ## 与小红书参考的差距
 
 - [ ] 整体氛围接近小红书产品海报
 - [ ] 主视觉占据首屏主要面积
 - [ ] UI 不抢主视觉
 - [ ] 截图后像 App 宣传图，不暴露工程化痕迹
