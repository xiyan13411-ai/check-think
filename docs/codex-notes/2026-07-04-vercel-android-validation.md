## 4. 真机验收执行建议

部署成功后，请按这个顺序验收：

1. 安卓 Chrome 打开 Vercel 预览链接。
2. 点击「重置测试数据」恢复初始状态。
3. 从 0 元开始测试快捷金额。
4. 使用大额自定义金额测试多碎片解锁。
5. 刷新页面检查数据持久化。
6. 关闭浏览器重新打开检查数据。
7. 测试成就节点。
8. 尝试添加到主屏幕。
9. 按 `docs/ANDROID_TEST_REPORT_TEMPLATE.md` 填一份报告。
10. 根据报告整理下一批修复任务。

建议重点观察：

- 存钱后碎片变化是否一眼明显
- 金额变化是否有爽感
- 拼图动画是否太快或太弱
- 文案是否适合截图
- 首页是否有点像 App，而不是普通网页
- 底部按钮是否被安卓系统导航遮挡
