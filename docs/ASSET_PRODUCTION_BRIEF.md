 # 视觉资产生产说明
 
 本文件定义「愿望拼图存钱」主视觉资产的生产规范。
 
 ---
 
 ## 目标视觉风格
 
 - 抽象智能手机背面（非正面屏幕）
 - 白色/玫瑰金/奶油色渐变背板
 - 有镜头模组（2-3 个镜头）
 - 有金属边框
 - 有玻璃/陶瓷材质反光
 - 柔和、温暖、小红书产品图氛围
 - 不出现 Apple/iPhone 官方 logo 或素材
 
 ## 资产清单
 
 | 文件 | 用途 | 建议尺寸 |
 |------|------|---------|
 | `phone-base.png` | 手机底图（未完成状态） | 1000×1600 px |
 | `phone-complete.png` | 手机完整图（100% 时显示） | 1000×1600 px |
 | `phone-shadow.png` | 手机阴影层（可选） | 1000×1600 px |
 | `fragments/fragment-01.png` – `fragment-14.png` | 碎片素材 | 每块 100–250 px |
 
 ## 碎片规范
 
 - 透明背景 PNG
 - 每块碎片是手机的一部分（有边缘厚度、高光、阴影）
 - 不要像普通色块或 emoji
 - 碎片拼合后还原完整的手机背面
 - 碎片之间保留微小缝隙（模拟裂开效果）
 
 ## 坐标系
 
 碎片坐标在 `lib/wish-assets.ts` 中配置，以**手机中心为原点 (0,0)**：
 
 - `targetX/Y`：碎片拼合到手机上时的位置
 - `startX/Y`：碎片从外部飞入时的起始位置
 - `unlockAt`：解锁进度阈值（0.05–1.0，间隔约 0.07）
 
 ## 替换资产
 
 1. 素材放入 `public/wish-assets/phone/` 对应目录
 2. 命名与 `lib/wish-assets.ts` 中的 `src` 路径一致
 3. 如果碎片数量/位置有变化，同步更新配置
 4. 素材缺失时自动使用 SVG fallback
