# 安卓真机 MVP 验收清单

> 项目：愿望拼图存钱 App
> 目标设备：Android Chrome
> 当前阶段：Post-MVP 验收

---

## 1. 基础运行

- [ ] 手机浏览器能打开页面
- [ ] 页面没有横向滚动
- [ ] 首屏视觉正常
- [ ] 页面没有明显白屏或闪烁
- [ ] 按钮足够好点
- [ ] 输入金额时弹出数字键盘
- [ ] 底部内容没有被系统手势栏遮挡

---

## 2. 存钱流程

- [ ] 点击 ¥10 正常增加金额
- [ ] 点击 ¥50 正常增加金额
- [ ] 点击 ¥100 正常增加金额
- [ ] 点击 ¥500 正常增加金额
- [ ] 自定义金额正常提交
- [ ] 输入 0 不提交
- [ ] 输入负数不提交
- [ ] 输入空值不提交
- [ ] 快速连续点击不会导致明显异常
- [ ] 当前金额不会显示 NaN

---

## 3. 反馈体验

- [ ] 金额变化明显
- [ ] 进度条变化明显
- [ ] 拼图变化一眼能看到
- [ ] 新增碎片动画准确（scale + rotate + opacity）
- [ ] 一次解锁多块碎片时，多块碎片依次动画（stagger）
- [ ] 刷新页面后不会重新播放全部碎片动画
- [ ] 存钱后出现鼓励文案
- [ ] 安卓震动反馈可用
- [ ] 动画不卡顿

---

## 4. localStorage 持久化

- [ ] 刷新页面后金额不丢
- [ ] 刷新页面后历史记录不丢
- [ ] 关闭浏览器再打开数据还在
- [ ] 重置测试数据后恢复默认状态
- [ ] 重置后刷新页面仍保持默认状态

---

## 5. 历史记录

- [ ] 每次存钱生成一条小票记录
- [ ] 记录显示金额
- [ ] 记录显示时间
- [ ] 记录显示文案
- [ ] 首页只展示最近 5 条
- [ ] 刷新后记录不丢

## 6. 成就弹窗

- [ ] 达到 25% 弹出成就
