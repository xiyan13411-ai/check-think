# Generated Assets Directory
Place AI-generated or designer-created visual assets here.

## Priority
- phone-render.png takes priority over phone-render.svg
- If PNG is present, FragmentedPhoneScene loads it automatically
- If PNG fails to load, falls back to built-in SVG

## Files
| File | Description | Recommended Size |
|------|-------------|-----------------|
| phone-render.png | Main phone back (transparent BG) | 1000×1600 px |
| phone-complete.png | Complete phone for 100% state | 1000×1600 px |
| fragments/fragment-*.png | Individual fragment assets (optional) | varies |

## Rules
- Transparent background only
- No watermarks, logos, or brand marks
- No Apple/iPhone official assets
- Single phone object per image
- Three-quarter or flat back view
